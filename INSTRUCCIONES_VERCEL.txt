OPCIÓN 1: VIA DASHBOARD DE VERCEL (MÁS FÁCIL) ⭐
═════════════════════════════════════════════════

Paso 1: Preparar el Repositorio Git
   □ Hacer commit de todos los cambios
   □ Pushear a GitHub/GitLab/Bitbucket

   Comandos:
   $ git add .
   $ git commit -m "Deploy: Club de Legumbres - Plataforma VIP"
   $ git push origin main

Paso 2: Ir a Vercel
   □ Visitar: https://vercel.com
   □ Iniciar sesión (usar las mismas credenciales del proyecto anterior)
   □ Click en "Add New Project" o "New Project"

Paso 3: Importar Repositorio
   □ Buscar el repositorio "club-legumbres" (o como lo hayas nombrado)
   □ Click en "Import"

Paso 4: Configurar Proyecto
   
   Framework Preset: Other (dejar en "Other")
   Root Directory: ./ (raíz del proyecto)
   Build Command: (dejar vacío - no necesita build)
   Output Directory: landing
   Install Command: (dejar vacío)

Paso 5: Deploy
   □ Click en "Deploy"
   □ Esperar 1-2 minutos
   □ ¡Listo! Tu sitio estará en: https://tu-proyecto.vercel.app


OPCIÓN 2: VIA CLI DE VERCEL (AVANZADO) 💻
═══════════════════════════════════════════

Paso 1: Instalar Vercel CLI
   $ npm install -g vercel

Paso 2: Login
   $ vercel login

   (Usar las mismas credenciales del proyecto anterior)

Paso 3: Deploy desde la raíz del proyecto
   $ cd /workspace
   $ vercel

   Responder:
   - Set up and deploy? → YES
   - Which scope? → Tu cuenta
   - Link to existing project? → NO
   - What's your project's name? → club-legumbres
   - In which directory is your code located? → ./

Paso 4: Deploy a Producción
   $ vercel --prod

   ¡Listo! Se mostrará la URL de tu sitio.


OPCIÓN 3: SCRIPT AUTOMATIZADO (ULTRA RÁPIDO) ⚡
═══════════════════════════════════════════════

   $ chmod +x deploy.sh
   $ ./deploy.sh

   El script hace todo automáticamente!

          (sensible a mayúsculas/minúsculas)

SOLUCIÓN: Verificar que estés en la raíz del proyecto
          Verificar que landing/index.html exista
